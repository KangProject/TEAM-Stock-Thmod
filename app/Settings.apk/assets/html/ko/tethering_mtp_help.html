USB 테더링

휴대폰을 USB 케이블로 컴퓨터에 테더링하여, 휴대폰의 인터넷 연결을 컴퓨터와 공유할 수 있습니다.
  • USB 드라이버를 PC Companion을 통해 설치하면, Windows Vista, Windows 7 및 Linux에서 테더링이 가능합니다.

  • 컴퓨터의 휴대폰 메모리 카드는 대용량 저장소 또는 미디어 전송 모드에 연결할 수 없습니다.

  • 그 밖의 운영체제와 USB 테더링을 사용하는 방법에 대한 자세한 내용은 http://www.android.com/tether#usb를 참조하십시오.

휴대용 Wi-Fi 핫스팟

휴대폰을 휴대용 Wi-Fi 핫스팟 모드로 설정하여, 휴대폰의 인터넷 연결을 한 대 이상의 컴퓨터 또는 기타 장치와 공유할 수 있습니다.
  • 휴대폰에서 Wi-Fi 핫스팟 기능을 사용하는 경우, 휴대폰의 응용프로그램을 해당 Wi-Fi 연결을 통해 인터넷에 액세스할 수 없습니다.

  • 핫스팟을 Wi-Fi 핫스팟 설정으로 구성할 수 있습니다.

  • 자세한 내용은 http://www.android.com/tether#wifi를 참조하십시오.
