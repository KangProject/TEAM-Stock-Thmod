Peranggitan USB (USB tethering)

Anda dapat meranggit telepon ke komputer menggunakan kabel USB, untuk berbagi sambungan Internet telepon dengan komputer
  • Peranggitan dapat dilakukan dengan Windows Vista, Windows 7, dan Linux

  • Anda tidak dapat memasang kartu SD telepon pada komputer ketika USB sedang diranggit

  • Untuk mendapatkan informasi selengkapnya termasuk cara menggunakan peranggitan USB pada SO lain, lihat http://www.android.com/tether#usb

Hotspot portabel Wi-Fi

Anda dapat mengaktifkan telepon menjadi hotspot Wi-Fi portabel untuk membagikan sambungan internet telepon dengan salah satu atau beberapa komputer atau perangkat yang lain
  • Saat telepon Anda berfungsi sebagai hotspot Wi-Fi, Anda dapat menggunakan aplikasi telepon untuk mengakses Internet melalui sambungan Wi-Fi-nya

  • Anda mengonfigurasikan hotspot dengan setelan hotspot Wi-Fi

  • Untuk informasi selengkapnya, kunjungi http://www.android.com/tether#wifi
