Tethering USB

Vous pouvez raccorder votre téléphone à votre ordinateur avec un câble USB, pour partager la connexion Internet de votre téléphone avec l'ordinateur
  • Le Tethering fonctionne sous Windows Vista, Windows 7 et Linux après avoir installé les pilotes USB à l'aide de PC companion

  • Vous ne pouvez pas connecter la carte mémoire de votre téléphone à votre ordinateur en mode Mémoire de masse ou Transfert de médias

  • Pour plus d'informations, y compris sur l'utilisation du Tethering USB avec d'autres systèmes d'exploitation, visitez http://www.android.com/tether#usb

Point d'accès Wi-Fi mobile

Vous pouvez transformer votre téléphone en point d'accès Wi-Fi mobile pour partager la connexion Internet de votre téléphone avec un ou plusieurs ordinateurs, ou bien encore d'autres appareils
  • Lorsque vous vous servez de votre téléphone en tant que point d'accès Wi-Fi, vous ne pouvez pas utiliser ses applications d'accès à Internet avec sa connexion Wi-Fi

  • Vous configurez le point d'accès avec les paramètres de point d'accès Wi-Fi

  • Pour plus d'informations, visitez http://www.android.com/tether#wifi
