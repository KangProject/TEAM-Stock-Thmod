Partage de connexion USB

Vous pouvez raccorder votre téléphone à votre ordinateur via un cordon USB pour partager la connexion Internet de votre téléphone avec votre ordinateur
  • Le partage de connexion est possible sur les systèmes d'exploitation Windows Vista, Windows 7 et Linux, après installation des pilotes USB par PC companion

  • Vous ne pouvez connecter la carte mémoire du téléphone à l'ordinateur sous les modes Stockage de masse et Transfert de médias

  • Pour obtenir davantage d'informations sur le partage de connexion USB ainsi que l'utilisation de celui-ci sous les autres systèmes d'exploitation, visitez http://www.android.com/tether#usb
