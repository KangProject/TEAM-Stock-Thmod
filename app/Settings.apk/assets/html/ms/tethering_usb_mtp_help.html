Penghubungan USB

Anda boleh menghubungkan telefon anda ke komputer anda melalui kabel USB, untuk berkongsi sambungan internet telefon anda dengan komputer anda
  • Kerja penghubungan dengan Windows Vista, Windows 7, dan Linux selepas anda telah memasangkan pemacu USB melalui PC companion

  • Anda tidak boleh menyambung kad ingatan telefon anda pada komputer anda dalam Simpanan massa atau mod Pemindahan media

  • Untuk mendapatkan maklumat lanjut, termasuk menggunakan penghubungan USB dengan OS lain, lihat http://www.android.com/tether#usb
