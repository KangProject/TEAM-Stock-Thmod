USB na tethering

Maaari mong i-tether ang iyong telepono sa iyong computer gamit ang isang USB na kable, upang ibahagi ang internet connection ng telepono sa iyong computer
  • Gumagana ang tethering sa Windows Vista, Windows 7, at Linux pagkaraang ma-install mo ang mga USB driver sa pamamagitan ng PC companion

  • Hindi mo maikokonekta ang memory card ng iyong teleopno sa iyong computer sa Mass storage o mode na Media Transfer

  • Para sa higit na impormasyon, kabilang ang paggamit ng USB na tethering sa ibang mga OS, tingnan ang http://www.android.com/tether#usb

Portable na Wi-Fi hotspot

Maaari mong gawing portable na Wi-Fi hotspot ang iyong telepono, upang maibahagi ang internet connection ng iyong telepono sa isa o higit pang mga computer o mga aparato
  • Kapag nagsisilbi bilang Wi-Fi hotspot ang iyong telepono, hindi mo magagamit ang mga application ng iyong telepono upang ma-access ang internet sa pamamagitan ng Wi-Fi na connection nito

  • Iko-configure mo ang hotspot gamit ang Wi-Fi hotspot na mga setting

  • Para sa higit na impormasyon, bisitahin ang http://www.android.com/tether#wifi
