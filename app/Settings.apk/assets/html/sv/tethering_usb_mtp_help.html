USB-tjudring

Du kan tjudra din telefon till datorn med en USB-kabel för att dela telefonens Internetanslutning med din dator
  • När du har installerat USB-drivrutinerna via PC Companion fungerar tjudring med Windows Vista, Windows 7 och Linux

  • Du kan inte ansluta ditt telefonminneskort på datorn i läget Masslagring eller Medieöverföring

  • För mer information, inklusive hur du använder USB-tjudring med andra operativsystem, gå till http://www.android.com/tether#usb
