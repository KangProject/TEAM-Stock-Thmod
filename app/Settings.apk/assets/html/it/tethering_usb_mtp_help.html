Tethering USB

È possibile collegare il telefono al computer con un cavo USB per condividere la connessione Internet del telefono con il computer.
  • Il tethering funziona con Windows Vista, Windows 7 e Linux, dopo l'installazione dei driver USB tramite PC companion

  • Non è possibile collegare la memory card del telefono con il computer in modalità Archiviazione di massa o Trasferimento multimediale

  • Per maggiori informazioni, incluse quelle sull'utilizzo del tethering USB con altri SO, consultare http://www.android.com/tether#usb
