Tambangan USB

Sampeyan bisa nambangake telpon menyang komputer nganggo kabel USB, bisa nuduhake sambungan internet telpon nganggo komputer
  • Tambangan bisa difungsekake nganggo Windows Vista, Windows 7, lan Linux sawise sampeyan instal driver USB liwat PC companion

  • Sampeyan ora bisa nyambungake kertu memori telpon ing komputer ing mode panyimpenan Gedhe utawa Transter Media

  • Kanggo informasi sing luwih lengkap, kalebu tambangan USB karo OSs liyane, bukak http://www.android.com/tether#usb
