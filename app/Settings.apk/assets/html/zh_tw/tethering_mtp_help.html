USB 網路共享

您可使用 USB 連接線讓手機和電腦共享網路，使您的電腦共享手機的網際網路連線
  • 在您透過 PC Companion 安裝 USB 驅動程式後，就可在 Windows Vista、Windows 7 和 Linux 上使用網路共享功能

  • 您無法在大量儲存裝置或媒體傳輸模式中將手機記憶卡連接至電腦

  • 如需詳細資訊，包括在其他作業系統上使用 USB 網路共享功能，請參見 http://www.android.com/tether#usb

可攜式 WLAN 作用點

您可將手機變成可攜式 WLAN 作用點，與一或多部電腦或其他裝置共享手機的網際網路連線
  • 當您將手機當成 WLAN 作用點使用時，就無法使用手機的應用程式透過其 WLAN 連線存取網際網路

  • 您可使用 WLAN 作用點設定來設定作用點

  • 如需更多資訊，請造訪 http://www.android.com/tether#wifi
