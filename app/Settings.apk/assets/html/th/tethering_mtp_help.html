การใช้ USB เชื่อมต่ออินเตอร์เน็ต

คุณสามารถเชื่อมต่อโทรศัพท์ของคุณเข้ากับคอมพิวเตอร์ได้ด้วยสาย USB เพื่อแบ่งปันการเชื่อมต่ออินเตอร์เน็ตของโทรศัพท์ให้แก่เครื่องคอมพิวเตอร์ของคุณ
  • การเชื่อมต่ออินเตอร์เน็ตจะสามารถใช้งานได้ร่วมกับ Windows Vista, Windows 7 และ Linux หลังจากที่คุณได้ติดตั้งไดร์ฟเวอร์ USB โดยใช้ PC companion แล้ว

  • คุณจะไม่สามารถเชื่อมต่อการ์ดหน่วยความจำของโทรศัพท์ที่เชื่อมต่อกับคอมพิวเตอร์ของคุณในโหมดการจัดเก็บข้อมูลหรือโหมดการโอนข้อมูลได้

  • สำหรับข้อมูลเพิ่มเติม รวมถึงการใช้ USB เชื่อมต่ออินเตอร์เน็ตกับ OS ชนิดอื่นๆ โปรดดูที่ http://www.android.com/tether#usb

Wi-Fi ฮอตสปอตแบบพกพา

คุณสามารถใช้โทรศัพท์ของคุณเป็น Wi-Fi ฮอตสปอตแบบพกพาเพื่อแบ่งปันการเชื่อมต่ออินเตอร์เน็ตของคุณกับเครื่องคอมพิวเตอร์หรืออุปกรณ์อื่นๆ หนึ่งเครื่องขึ้นไปได้
  • เมื่อโทรศัพท์ของคุณทำหน้าที่เป็น Wi-Fi ฮอตสปอต คุณจะไม่สามารถใช้โปรแกรมในโทรศัพท์ของคุณเพื่อเข้าสู่อินเตอร์เน็ตทางการเชื่อมต่อแบบ Wi-Fi ได้

  • คุณสามารถตั้งค่าฮอตสปอตได้ด้วยการตั้งค่า Wi-Fi ฮอตสปอต

  • สำหรับข้อมูลเพิ่มเติม โปรดไปที่ http://www.android.com/tether#wifi
