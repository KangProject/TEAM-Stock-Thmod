Ancoratge USB

Podeu ancorar el telèfon a l'ordinador amb un cable USB per compartir la connexió a Internet del telèfon amb l'ordinador
  • L'ancoratge funciona amb Windows Vista, Windows 7 i Linux després que instal·leu els servidors USB mitjançant PC companion

  • No podreu connectar la targeta de memòria del telèfon a l'ordinador en el mode d'Emmagatzematge massiu o de Transferència de fitxers multimèdia

  • Per obtenir més informació, inclosa la d'utilitzar l'ancoratge USB amb altres OS, vegeu http://www.android.com/tether#usb
