Привязка по USB

Можно выполнить привязку телефона к компьютера с помощью кабеля USB для использования интернет-подключения телефона на компьютере
  • Привязка работает с ОС Windows Vista, Windows 7 и Linux после установки драйверов USB через PC Companion

  • Нельзя подключить карту памяти телефона на компьютере в режиме хранения данных или передачи мультимедийных файлов

  • Для получения дополнительных сведений, включая привязку по USB при использовании других ОС, см. http://www.android.com/tether#usb

Портативная зона доступа Wi-Fi

Можно превратить телефон в портативную зону доступа Wi-Fi, чтобы использовать интернет-подключение телефона с одним или более компьютерами либо другими устройствами
  • Когда телефон служит зоной доступа Wi-Fi, нельзя использовать приложения телефона для доступа к Интернету через его соединение Wi-Fi.

  • Настройте зону доступа с помощью параметров зоны доступа Wi-Fi

  • Для получения дополнительной информации посетите веб-сайт http://www.android.com/tether#wifi
