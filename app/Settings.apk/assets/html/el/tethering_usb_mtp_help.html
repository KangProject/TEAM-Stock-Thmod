Χρήση του κινητού ως μόντεμ USB (tethering)

Μπορείτε να χρησιμοποιήσετε το κινητό σας τηλέφωνο ως μόντεμ για τον υπολογιστή σας με ένα καλώδιο USB, για να κάνετε κοινή χρήση με τον υπολογιστή σας της σύνδεσης του τηλεφώνου σας στο Internet
  • Η χρήση του κινητού ως μόντεμ USB (tethering) λειτουργεί με τα Windows Vista, τα Windows 7 και το Linux αφού εγκαταστήσετε τα προγράμματα οδήγησης USB μέσω της εφαρμογής PC Companion

  • Δεν μπορείτε να συνδέσετε την κάρτα μνήμης του τηλεφώνου σας στον υπολογιστή σε λειτουργία μαζικής αποθήκευσης ή μεταφοράς πολυμέσων

  • Για περισσότερες πληροφορίες, συμπεριλαμβανομένης της χρήσης του κινητού ως μόντεμ USB με άλλα λειτουργικά συστήματα, δείτε http://www.android.com/tether#usb
