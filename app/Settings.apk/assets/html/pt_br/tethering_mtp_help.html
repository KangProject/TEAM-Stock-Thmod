Redirecionamento por USB

Você pode redirecionar o telefone para o computador com um cabo USB, para compartilhar a conexão com a Internet do telefone com o computador
  • O redirecionamento funciona com Windows Vista, Windows 7 e Linux, assim que você tiver instalado os drivers USB por meio do PC Companion

  • Não é possível conectar o cartão de memória do telefone ao computador no modo Armazenamento maciço ou Transferência de mídia

  • Para obter mais informações, inclusive sobre como usar o redirecionamento por USB com outros sistemas operacionais, consulte http://www.android.com/tether#usb

Ponto de acesso Wi-Fi portátil

Você pode transformar seu telefone em um ponto de acesso Wi-Fi portátil para compartilhar a conexão do telefone à Internet com um ou mais computadores ou outros dispositivos
  • Enquanto o telefone funciona como ponto de acesso Wi-Fi, não é possível usar os aplicativos do telefone para acessar a Internet por meio de sua conexão Wi-Fi

  • Use as configurações de ponto de acesso Wi-Fi para configurar o ponto de acesso

  • Para obter mais informações, visite http://www.android.com/tether#wifi
