USB সংযোগ

আপনার ফোনের ইন্টারনেট সংযোগ কম্পিউটারের সাথে অংশীদারি করতে একটি USB কেবলের মাধ্যমে আপনি আপনার ফোনটিকে কম্পিউটারের সাথে সংযোগ করতে পারেন
  • PC সহযোগীর মাধ্যমে আপনি USB ড্রাইভারগুলি সংস্থাপন করার পরে Windows Vista, Windows 7, এবং Linux-এর সাথে সংযোগের কাজটি করে

  • আপনি বিপুল সংরক্ষণ বা মিডিয়া স্থানান্তর মোডে আপনার কম্পিউটারে আপনার ফোন মেমরি কার্ডটি সংযোগ করতে পারবেন না

  • অন্যান্য OS-এর সাথে USB সংযোগের ব্যবহার সহ আরও তথ্যের জন্য, http://www.android.com/tether#usb দেখুন

পোর্টেবেল Wi-Fi হটস্পট

আপনার ফোনের ইন্টারনেট সংযোগ একটি বা তার বেশি কম্পিউটারের সাথে বা যন্ত্রের সাথে অংশীদারি করতে আপনি নিজের ফোনটিকে একটি পোর্টেবেল Wi-Fi হটস্পটে পরিবর্তিত করতে পারেন
  • আপনার ফোন যখন একটি Wi-Fi হটস্পট হিসাবে কাজ করে তখন, সেটির Wi-Fi সংয়োগের মাধ্যেমে ইন্টারনেট অ্যাক্সেস করার জন্য আপনি ফোনের অ্যাপ্লিকেশন ব্যবহার করতে পারবেন না

  • Wi-Fi হটস্পট সেটিংসের মাধ্যমে আপনি হটস্পট কনফিগার করুন

  • আরও তথ্যের জন্য, দেখুন http://www.android.com/tether#wifi
