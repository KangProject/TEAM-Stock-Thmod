USB সংযোগ

আপনার ফোনের ইন্টারনেট সংযোগ কম্পিউটারের সাথে অংশীদারি করতে একটি USB কেবলের মাধ্যমে আপনি আপনার ফোনটিকে কম্পিউটারের সাথে সংযোগ করতে পারেন
  • PC সহযোগীর মাধ্যমে আপনি USB ড্রাইভারগুলি সংস্থাপন করার পরে Windows Vista, Windows 7, এবং Linux-এর সাথে সংযোগের কাজটি করে

  • আপনি বিপুল সংরক্ষণ বা মিডিয়া স্থানান্তর মোডে আপনার কম্পিউটারে আপনার ফোন মেমরি কার্ডটি সংযোগ করতে পারবেন না

  • অন্যান্য OS-এর সাথে USB সংযোগের ব্যবহার সহ আরও তথ্যের জন্য, http://www.android.com/tether#usb দেখুন
