USB 绑定

您可以通过 USB 连接线将手机与计算机绑定，从而与计算机共享手机的互联网连接
  • 通过 PC Companion 安装好 USB 驱动程序后，绑定可适用于 Windows Vista、Windows 7 和 Linux

  • 您无法以海量存储或媒体传送模式在计算机上连接手机记忆卡

  • 要了解详细信息（包括如何在其他操作系统上使用 USB 绑定），请访问 http://www.android.com/tether#usb

便携式 WLAN 热点区

您可以将手机设为便携式 WLAN 热点区，从而与一台或多台计算机或者其他设备共享手机的互联网连接
  • 将手机用作 WLAN 热点区后，您就无法通过其 WLAN 连接使用手机的应用程序访问互联网了

  • 您可以使用 WLAN 热点区设定配置热点区

  • 要了解详细信息，请访问 http://www.android.com/tether#wifi
