שימוש בטלפון כמודם סלולרי באמצעות כבל USB

באפשרותך להשתמש בטלפון כמודם סלולרי עבור המחשב באמצעות כבל USB, כדי לשתף את החיבור לאינטרנט עם המחשב
  • שימוש בטלפון כמודם סלולרי נתמך במערכות ההפעלה Windows Vista‏, Windows 7 ו-Linux לאחר התקנת מנהלי התקן USB באמצעות PC companion

  • באפשרותך לחבר את כרטיס הזיכרון בטלפון אל המחשב במצב Mass storage או Media Transfer

  • לקבלת מידע נוסף, כולל שימוש במחשב כמודם סלולרי באמצעות USB במערכות הפעלה אחרות, עיין באתר http://www.android.com/tether#usb
